### Reporting Bugs

If you find a bug, please open an [Issue](https://github.com/XedDexTools/xed-solana-metadata/issues) with:
- A clear description of the bug
- Steps to reproduce
- Expected vs. actual behavior
- Screenshots (if applicable)
- Your environment (OS, Node version, etc.)

### Suggesting Features

We welcome feature suggestions! Open an [Issue](https://github.com/XedDexTools/xed-solana-metadata/issues) with:
- A clear description of the feature
- Why it would be valuable
- Potential implementation approach (if you have ideas)

### Code Contributions

1. **Fork the repository**
2. **Create a feature branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes**
   - Follow the existing code style
   - Add comments for complex logic
   - Update documentation if needed
4. **Test your changes**
   ```bash
   npm run build
   npm run dev
   ```
5. **Commit your changes**
   ```bash
   git commit -m "Add: description of your changes"
   ```
6. **Push to your fork**
   ```bash
   git push origin feature/your-feature-name
   ```
7. **Open a Pull Request**
   - Provide a clear description
   - Reference any related issues
   - Wait for review and feedback
